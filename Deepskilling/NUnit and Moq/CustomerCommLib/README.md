# CustomerCommLib

This repository contains the CustomerCommLib solution with unit tests (NUnit and Moq).

## Description

Projects included:
- CustomerCommLib - main library
- CustomerComm.Tests - unit tests

## How to run tests

Open the solution in Visual Studio and run the Test Explorer, or use dotnet test from the project folder:

```powershell
dotnet test
```
